# ColorSupport

## Description

**ColorSupport** provide system colors of iOS 13 to older iOS.

## Usage

```
let red = UIColor.systemRed
```

## Installation

### Carthage

To install it, simply add the following line to your Cartfile.

`github "noppefoxwolf/ColorSupport"`

## Requirements

- Swift5.0

- iOS10+
